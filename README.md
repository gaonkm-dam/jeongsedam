# 정세담 학습 시스템 설치 및 실행 가이드

## 1. 필수 패키지 설치

```powershell
pip install streamlit openai pandas
```

## 2. OpenAI API 키 설정

프로젝트 폴더에 `.env` 파일을 생성하고 다음 내용을 입력합니다:

```
OPENAI_API_KEY=your_actual_api_key_here
```

또는 환경 변수로 설정:

```powershell
$env:OPENAI_API_KEY="your_actual_api_key_here"
```

## 3. 실행

```powershell
streamlit run app.py
```

## 4. 로그인 정보

기본 제공 학생 계정:
- 학생1: student1 / pass1
- 학생2: student2 / pass2
- 학생3: student3 / pass3

## 5. 기능 목록

### 학생 기능
1. **로그인**: 3개 학생 계정
2. **대시보드**: 총 문제 수, 정답률, 최근 학습일, 레벨, 순위 표시
3. **심리 체크**: 20문항 설문 (결과는 학부모/교사만 조회)
4. **새로운 학습**:
   - 과목 선택 (국어, 영어, 수학, 과학, 사회, 역사, 한자)
   - 학년 선택 (초1~고3)
   - 교과서 페이지 입력
   - 난이도 선택 (쉬움/보통/어려움)
   - 시험 유형 선택 (중간/기말/모의/수능)
   - 문제 수 설정 (기본 10개)
5. **문제 풀이**:
   - 객관식/주관식/서술형 혼합 문제
   - 실시간 검색 기능 (과목별 단어/공식/개념)
   - 검색 내용 단어장 저장
6. **단어장**: 저장된 검색 내용을 과목별로 조회
7. **제출 결과**: 정답/오답 표시, 해설 제공, 점수 표시
8. **학습 이력**: 날짜별 학습 기록 및 문제 재확인
9. **순위**: 전체 학생 순위 (총점 기준)
10. **동기부여 문구**: 학습 시작/완료 시 자동 표시
11. **추천 도서**: 이달의 추천 도서 10권
12. **목표 대학**: 대학명/학과 설정 및 변경

## 6. 데이터베이스

SQLite 자동 생성 (student_system.db):
- students: 학생 정보
- study_sessions: 학습 세션
- questions: 문제 및 답안
- psychological_tests: 심리 체크 결과
- search_history: 검색 및 단어장
- rank_cache: 순위 캐시

## 7. 주의사항

- OpenAI API 키가 없으면 문제 생성, 검색, 동기부여, 도서 추천 기능이 작동하지 않습니다
- 기존 기능은 절대 삭제하지 마세요 (정책/투자용 MVP)
- 모든 기능은 추가만 가능합니다
