학생

-로그인 / 계정 3개로 로그인 해서 데이터 볼수있도록 계정 3개

-심리 체크 페이지/ 문항 20개(점수별로 학생에 상태를 학부모,교사가 볼수있어야함, 학생은 문제만 풀ㄹ고 심리 상태는 볼수없어야함)

-순위 / 문제를 풀면 풀었던 문제를 기반으로 학생들 순위를 나오게 해줌

-로그인 후 첫 화면 학생 문제 풀이 레벨 등등 통계 화면 나와줘야함

-최근 학습이력 / 특정 날짜 클릭하면 그날 풀었던 문제 나옴

-새로운 학습시작 / 과목(국어,영어,수학,과학,사회,역사,한자) / 학년 초1~고3까지 자유롭게 선택 가능 / 오늘의 학습교과서 (시작 페이지 선택, 종료페이지 선택)
  /난이도(쉬움,보통,어려움) / 시험 선택(중간고사,기말고사,모의고사,수능시험)

-문항 수 / 기본적으로 10개 부터 시작(문항 문제는 10개 이하, 이상 선택 할수있게 해야함)그거에 맞게 객관식,주관식,서술형 문제가 기본적으로 문항수가 변경이 되어야함
기본적인 숫자는 프로그램이 정해 주지만 사람이 별도로 선택도 할수있게 해야함

-문제 생성 시작 버튼 누르면 선택한 문항수에 문제가 나오면됨

-문제1번, 문제 2번 등등 문제(번호) 옆에 해당 과목에 맞는 단어 장이 나오면됨 예를 들어 영어는 단어+문법 / 수학은 공식 / 국어는 단어 / 한자는 한자뜻
이렇게 문제에서 이해가 안가는 부분이 있으면 바로 검색해서 정보를 찾아볼수있는 기능 필요
그리고 이렇게 찾아본 단어또는공식은 별도에 단어장이라는 항목으로 들어가서 그 단어장을 출력 또는 프로그램 화면에서 볼수있는 기능 필요

-문제를 풀고 제출하기 버튼을 누르면 오답과 정답이 나오는데 오답은 오답대로 해석, 정답은 정답대로 해석이 나와서 학생들이 
틀린문제든 맞은 문제든 이해 할수 있도록 기능 제공
채점 결과가 눈에 바로 보이도록 ui및 데이터 제공

-학생들이 힘든 학습을 포기 하지 않도록 동기부여 하는 문구가 문제 풀기 전후로 화면에 보이도록 매번 다른 동기부여 문구가 나와야함

-학생들 책잃는 습관이 중요해 이달에 책 추천이라는 항목이 있어서 10권정도 매달 추천 해주는 기능

-학생이 직접 가고싶은 대학+학과를 선택해서 그 목표를 향해서 갈수있는 기능 (자유롭게 변경 가능해야함)


이 외에 추가로 필요한 기능은 너가 추천해줘 추천해준걸 보고 내가 판단을 할테니까.
